## はじめに

**パルワールド (Palworld)** は、クリーチャー収集、サバイバル、基地建設を独自に組み合わせたゲームで世界中を席巻しています。ゴールドを増やしたい、パルのステータスを変更したい、レアアイテムをインベントリに追加したい場合、このガイドで Palworld のセーブファイルを安全に編集する方法を詳しく説明します。

無料オンライン **Palworld セーブエディタ** を使えば、怪しいソフトウェアをダウンロードすることなくゲームを簡単に変更できます。すべての処理はブラウザで行われ、セーブファイルはあなたのコンピュータから離れることはありません。

## セーブファイルの場所

### Windows (Steam)
```
%LocalAppData%\Pal\Saved\SaveGames\<SteamID>\
```

### Windows (Xbox/Game Pass)
```
%LocalAppData%\Packages\PocketpairInc.Palworld_<id>\SystemAppData\wgs\
```

### Steam Deck (Proton)
```
~/.steam/steam/steamapps/compatdata/1623730/pfx/drive_c/users/steamuser/AppData/Local/Pal/Saved/SaveGames/
```

### セーブファイルの構造

セーブフォルダ内には以下のファイルがあります：

| ファイル | 説明 |
|---------|------|
| `Level.sav` | ワールドデータ（基地建造物、野生パルなど） |
| `Players/<ID>.sav` | プレイヤーデータ（ステータス、インベントリ） |
| `LocalData.sav` | ローカル設定 |

**重要**：プレイヤーデータを変更するには、`Players` フォルダ内のファイルを編集します。

## ステップ 1：セーブをバックアップ

変更を加える前に、**必ずバックアップを作成**してください：

1. セーブフォルダに移動
2. フォルダ全体を安全な場所（デスクトップなど）にコピー
3. 日付を付けてラベル付け（例：`Palworld_Backup_2025年12月`）

## ステップ 2：オンラインエディタにアップロード

1. [Palworld セーブエディタ](/ja/editor/unreal)（Unreal Engine エディタ）を開く
2. プレイヤーの `.sav` ファイルをドラッグ＆ドロップ
3. GVAS パーサーがバイナリデータを処理するのを待つ

## ステップ 3：データを見つけて編集

解析が完了すると、すべてのゲームデータの JSON ツリーが表示されます。編集可能な項目：

### ゴールド/お金を編集
以下の名前のプロパティを探します：
- `Money`
- `Gold`
- `Currency`

数値を希望の金額に変更するだけです。

### パルのステータスを変更
`CharacterSaveParameterMap` に移動してパルを見つけます：
- **レベル**：パルのレベルを直接変更
- **ステータス**：HP、攻撃力、防御力を修正
- **パッシブスキル**：受動的能力を編集または追加
- **アクティブスキル**：アクティブスキルスロットを調整

### 編集可能な項目まとめ

| カテゴリ | 編集可能な内容 |
|---------|--------------|
| **お金** | ゴールド、通貨の量 |
| **パル** | レベル、ステータス、スキル、特性 |
| **インベントリ** | アイテム、装備、リソース |
| **プレイヤー** | レベル、HP、スタミナ、テクノロジーポイント |
| **基地** | 建設進捗、施設レベル |

## よくある質問

### マルチプレイヤー/専用サーバーで機能しますか？

**シングルプレイヤー** および **Co-op（ホストとして）**：はい、ローカルセーブファイルを編集できます。

**専用サーバー**：セーブはサーバー側に保存されています。変更するにはサーバーへのアクセスが必要です。

### セーブ編集で BAN されますか？

Palworld のシングルプレイヤー/Co-op にはアンチチートがありません。ただし、専用サーバーでは、管理者がチート禁止ルールを持っている場合があります。
